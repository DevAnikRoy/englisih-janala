Frequently Asked Questions

1. What is the difference between Var, Let & Constant ?
Ans: Var- We actually don't use var for the Modern Java Script. Coz if we want we can Reassignment this and it also allowed us to Redeclaration and also it make our code Lazy. Let- We use this in Modern JS. It allowed us Reassignment but not allowed us foe Redeclared. We mainly use it variable that change. Const- Const is mainly use for in Unchangeable position. And it's not allowed the Reassignment and the Redeclaration.
2. What is the difference between map{}, fetch{} and filter{}?
Ans: map{} - we mainly use map for use to transfer elements in an array. Creates a new array without changing the original. fetch{} - Used to make network requests to an API. A Promise that resolves to the Response object representing the response to the request. Commonly Used with .then() or async/await To handle the response. filter() - Used to create a new array with elements. Commonly use for Data filtering.
3. What are Arrow function and how are they different from regular functions?
Ans: Arrow functions are a concise way to write functions in JavaScript, introduced in ES6. They provide a shorter syntax compared to regular functions and have some differences in behavior, particularly with regard to the this keyword.
4. How do JS Promise work ?
Ans: Promises make asynchronous code easier to read and understand. They provide a structured way to handle errors in asynchronous operations. They allow us to chain asynchronous operations together, simplifying complex workflows.
5. How does closer work in JS ?
Ans: Closures allow you to create private variables and functions. They are essential for implementing patterns like function factories and event handlers.
